# legal-document-explainer

Skill perfeitamente estruturada para analisar documentos jurídicos, traduzi-los para linguagem simples e identificar cláusulas abusivas.

## Estrutura do Projeto
- `SKILL.md`: Instruções principais para o modelo de IA.
- `scripts/`: Scripts auxiliares de extração e processamento.
- `references/`: Bases de conhecimento sobre riscos e glossário jurídico.
- `assets/`: Templates para geração de relatórios.

Desenvolvido para empoderar o consumidor e garantir transparência em contratos e termos de uso.
